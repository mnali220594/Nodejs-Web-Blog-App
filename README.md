# Nodejs-Web-Blog-App
A web blog that built with nodejs, mongodb, and edge templating engine

## Usage

Rename ".env.example" to ".env" and update the values/settings to your own

# Install all dependencies
```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

```
